# Athena2
Athena is a java library for interacting with the Fortnite/Epic Games API.

# Features
- Account lookup support
- In-game events support
- Shop support
- Friends support (and XMPP!)
- Stats V2
- ~Stats V1 (UPCOMING)
- ~Presence support (UNFINISHED)
- ~Party support (UPCOMING)
- And more...
